# SQL & PL/SQL Practice Project

This repository contains self-driven SQL and PL/SQL practice examples focused on query reasoning, debugging, and database logic. The project demonstrates hands-on experience with complex SQL queries, PL/SQL stored procedures, and exception handling, aligned with real-world problem statements.

## Files Included

- **complex_queries.sql**  
  Contains complex SQL queries using joins, subqueries, aggregations, filtering, and sorting to retrieve and analyze relational data.

- **stored_procedures.sql**  
  Includes PL/SQL stored procedures demonstrating parameterized procedures, transactional operations, and basic dynamic SQL usage.

- **exception_handling.sql**  
  Demonstrates PL/SQL exception handling using predefined and generic exceptions to manage runtime errors and ensure safe execution.

## Key Concepts Covered

- Complex SQL queries  
- Joins, subqueries, and aggregations  
- PL/SQL stored procedures and functions  
- Exception handling  
- Query debugging and validation  
- Basic dynamic SQL  

## Tech Stack

- SQL  
- PL/SQL  
- Oracle SQL / MySQL  

## Purpose

This project is intended for learning and practice purposes and showcases foundational database skills required for roles involving SQL reasoning, query analysis, and data validation, including LLM Trainer and database-focused roles.
